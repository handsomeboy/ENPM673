Steps to run the program
1)	First run Demosaic.m script, it will make color images from the Bayer images in the code folder. The script will need the path of the Bayer images.
2)	Then run the code.m script. The script will need the path of camera models and bayer images to estimate the camera parameters.


Note- The converted color images should be in the code folder for successful execution of the program.
